The basic structure of an html document is as follows:

    This is a paragraph! We start each paragraph with a " < p >" and end it with a "< / p >"" tag!


  1. "< html >"
  2. "< head >"
  3. "< title >"
  4. "< body >"

    5. Each one of these HTML tags is followed by a closing tag "< / >"

    • regular text
    • bold text
    • italicized text
    • superscript text

Everything inbetween an opening and closing html "tag" is part of the content of that element.
